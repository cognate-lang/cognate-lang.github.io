Def Fizzbuzz

   Case (Multiple of 3)  is "fizz"
   Case (Multiple of 5)  is "buzz"
   otherwise ();

Cognate is a project aiming to create a human readable programming language with as little syntax as possible. Where natural language programming usually uses many complex syntax rules, instead Cognate takes them away. What it adds is simple, a way to embed comments into statements. 
~~ Towers of Hanoi in Cognate

Def Move discs as (

   Let N be number of discs;
   Let A be first rod;
   Let B be second rod;
   Let C be third rod;

   Unless Zero? N (
      Move - 1 N discs from A via C to B;
      Prints ("Move disc " N " from " A " to " C);
      Move - 1 N discs from B via A to C;
   )
);

As you can see, Cognate ignores words starting with lowercase letters, allowing them to be used to describe functionality and enhance readability. Cognate is a stack-oriented programming language like Forth or Factor, except statements are evaluated in reverse. This gives the benefits of concatenative programming and the readability of prefix notation.
  • Point-free functions
  • Operation chaining
  • Multiple return values
  • Combinator oriented programming
As well as these, Cognate offers unique features of its own.
  • Predicate pattern matching
  • Natural language programming
  • Gradual typing with inference
